# AWS S3 Channel

## Overview
Simulates data exfiltration via an AWS S3 PUT object operation.

## Configuration
```yaml
channels:
  s3:
    enabled: true
    bucket: "your-test-bucket"
    region: "us-east-1"
    access_key: "AKIA..."
    secret_key: "..."
```

## Requirements
- No root required
- Bucket must permit PutObject from test machine
- Uses AWS SigV4 signing (no SDK dependency)
